# GrannyLegacy---Trainer

- Use the 7.2 Melon Loader version.
- Boot the game and close it again (for creating folders and configs)
- Then put the [trainer.dll](https://github.com/Trofline/GrannyLegacy---Trainer/releases/download/1.0/GrannyLegacy-Trainer.dll) into the mod folder
- Boot the game and bring out the menu by pressing the [INSERT] key
